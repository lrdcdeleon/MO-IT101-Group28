# MotorPH Payroll System - Milestone 2

## Project Overview
This repository contains the development progress for **Phase 1** of the MotorPH Employee Payroll System. The primary goal of this phase is to automate the calculation of weekly hours worked, gross salary, and standard government deductions to ensure accuracy and efficiency.

## Milestone 2 Features
* **Modular Architecture**: The system is decomposed into a main application class (`MotorPH_App`) and a dedicated logic class (`PayrollCalculator`) for high reusability.
* **Automated Computations**:
    * **Gross Weekly Salary**: Calculated based on hourly rates and weekly hours.
    * **Government Deductions**: Automated calculation for **SSS**, **PhilHealth**, and **Pag-IBIG**.
    * **Net Weekly Salary**: Final take-home pay computation after all deductions.
* **Inline Documentation**: Comprehensive Javadoc and inline comments explain the logic and formulas used.

## Technical Requirements
* **Language**: Java
* **IDE**: Apache NetBeans
* **Version Control**: Git/GitHub

## How to Run
1. Clone this repository to your local machine.
2. Open the project in Apache NetBeans.
3. Locate `MotorPH_App.java` in the `moit101.group28` package.
4. Right-click and select **Run File** to view the payroll summary output.
